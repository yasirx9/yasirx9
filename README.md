## **About Me**

I'm a **Software Engineering Undergraduate** (6th Semester) at **Sindh Madressatul Islam University (SMIU)**.

I build applied **Machine Learning** projects — mostly around **text and document classification** — and I'm sharpening full-stack skills on the side.

**I work across 3 areas:**

**1. Text Classification** — models that read text and predict tone, category, or intent.
**2. Applied ML Pipelines** — cleaning data, training classical ML/NLP models, and evaluating them properly.
**3. Small full-stack tools** — Java/JS apps that turn ideas into something usable.

I'm currently leveling up my ML fundamentals while picking up more software engineering practice through coursework and side projects.
